BUILD INSTRUCTIONS

* This installer was built with Inno Setup 3 which can be downloaded here:
  http://www.jrsoftware.org/isinfo.php

* Please run "FetchFiles.bat" Before compiling the installer. This script
  collects needed files and libraries.

* To create the installer build the Release version of RapidSVN, 
  open the rapidsvn.iss from the packages/win32 directory, then 
  click Compile.
